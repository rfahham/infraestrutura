# Infraestrutura

Repositório dedicado aos projetos e automações relacionados à infraestrutura da plataforma, incluindo provisionamento, configuração e gerenciamento de ambientes.

**Tecnologias e ferramentas utilizadas:**

* **Terraform** – Provisionamento e gerenciamento de infraestrutura como código (IaC).
* **Ansible** – Automação de configuração, implantação e administração de servidores.
